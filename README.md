# Напольный вентилятор на ИК пульте (NEC)

В данном репозитории три версии вентилятора:  
Версия **"ver 0.x"** была когда только начал заниматься изучением Arduino и является "громоздкой" и неудобной.  
Версия **"ver 1.x"** получилась проходной, не очень удачной.  
Версия **"ver 2.x"** получилась самой компактной и стабильной, к тому же добавлена 4-я (пониженная) скорость вентилятора, больше дорабатывать особо нечего в ней.  


Проект делал лично для себя, выкладываю его сюда что бы не потерялся и может кто захочет повторить.
